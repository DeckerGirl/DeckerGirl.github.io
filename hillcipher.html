Hill Cipher File Encryption Program

Description

This Python program encrypts alphabetic letters in a file using the Hill cipher. Users provide a plaintext file and a key matrix file via command-line arguments, and the program outputs the encrypted ciphertext. Non-alphabetic characters are ignored.

Skills Learned

  • Python programming and file handling
  • Matrix operations and modular arithmetic
  • Algorithm design and debugging
  • Cryptography concepts and Hill cipher implementation
