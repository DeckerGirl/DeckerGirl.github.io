Destinease - Travel Website

Group Project: Developed collaboratively with classmates to build a full-featured travel planning tool.

Description

Destinease is a collaborative travel website that allows users to plan trips efficiently. It builds personalized travel budgets, schedules, and includes integrated flight and hotel booking functionality. The project emphasized teamwork, modular design, and real-world web application development practices.

Skills Learned

  • Front-end web development: HTML, CSS, JavaScript
  • Back-end integration with APIs for flights and hotels
  • Collaborative software development and version control (Git)
  • Project management and task delegation within a team
  • User experience (UX) design and responsive layout implementation
